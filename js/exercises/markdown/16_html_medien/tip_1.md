# Video auf 100% der verfügbaren Breite setzen

Um das Video auf 100% der verfügbaren Breite anzuzeigen benötigst du das Attribut width="100%".

## Beispiel
```html
<video id="video1" width="100%">
    <source src="..." type="...">
</video>
```
