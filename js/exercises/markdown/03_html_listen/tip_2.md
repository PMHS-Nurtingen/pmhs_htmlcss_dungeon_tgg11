# Listen erstellen
Eine geordnete Liste wird mit dem **ol**-Element (OrderedList) erstellt, eine ungeordnete mit dem **ul**-Element (UnorderedList). In diesem Element werden dann Listenelemente mit dem **li**-Element (ListItem) erstellt.
