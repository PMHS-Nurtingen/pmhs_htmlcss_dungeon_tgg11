# Typselektoren
Typselektoren geben Styles für bestimmte HTML-Tags an, will man z.B. für alle ListItem (li) Elemente einen Style angeben schreibt man als Selektor einfach den HTML-Elementnamen *`li { ... }`* verwenden.
