# Auszeichnungen verwenden
Innerhalb eines Paragraphen können unterschiedliche Auszeichnungen verwendet werden, diese liegen meist in Form weiterer HTML-Elemente (wie z.B. ) *`<p>Das ist ein <strong>Text</strong></p>`*. Unter dem Link findest du weitere Auszeichnungsmöglichkeiten.
